# Plan de Ilustraciones Tradicionales y Retro para Crónicas del Nexo

Este plan detalla la lista de las próximas 20 ilustraciones a generar, utilizando exclusivamente los estilos tradicionales y artísticos del juego base (evitando el estilo de "diseño digital/render 3D" moderno) para mantener la estética clásica de cartas físicas coleccionables.

## Estilos Artísticos de Referencia
*   **Arte de Píxeles 16-Bit**: Estilo retro de juego de consola.
*   **Acuarela Mística**: Trazos suaves de acuarela con tonos degradados mágicos.
*   **Boceto a Carbón**: Carboncillo dinámico con trazos en blanco y negro de alta intensidad.
*   **Pintura al Óleo**: Capas de óleo densas con texturas rústicas y clásicas.
*   **Grabado Medieval en Madera (Woodcut)**: Grabados de tinta negra estilo grimorio antiguo.
*   **Novela Gráfica (Cel-Shading)**: Bordes de tinta definidos y sombreados por bloques de color.
*   **Vidriera Gótica (Stained Glass)**: Mosaicos translúcidos iluminados desde atrás.
