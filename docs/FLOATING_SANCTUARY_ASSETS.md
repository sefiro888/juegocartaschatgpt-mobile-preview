# Recursos del Santuario Flotante

Fecha de incorporación: 2026-07-11

## Geometría original

Toda la arquitectura principal de `floating-sanctuary.glb` es original de este proyecto. Se genera mediante:

- `tools/blender/build_floating_sanctuary.py`
- `art/blender/floating-sanctuary.blend`

No se ha incorporado ningún modelo 3D de terceros. El archivo incluye plataformas, puentes, cimientos, columnas, portal, medallones, cristales y ornamentación dorada.
